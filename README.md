# Proyecto Homeland

## Descripción
Página web responsiva que presenta lugares históricos y destinos de viaje alrededor del mundo. El proyecto está desarrollado siguiendo las especificaciones de diseño de Figma y utiliza la metodología BEM para la organización del código CSS.

## Tecnologías utilizadas
- HTML5
- CSS3
- Metodología BEM
- Diseño responsivo con media queries
- Figma (para especificaciones de diseño)

## Funcionalidades
- Diseño completamente responsivo
- Navegación intuitiva
- Galería de imágenes
- Información detallada de destinos

## Cómo visualizar el proyecto
1. Clona este repositorio
2. Abre el archivo `index.html` en tu navegador
3. ¡Disfruta explorando los destinos!
